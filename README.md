 SLDHA – Sierra Leone Digital Health Mobile App
 Project Overview
This project designs a mobile‑first health application for Sierra Leone Digital Health Authority.
Goal: make health services easier, faster and accessible for everyone.
Also suitable for database and system design assignments.

 Features & Structure
 1. User Entry Flow
- Get Started / Welcome screen
- New User Registration
- Secure Login

2. Main Functions
- Dashboard – main home menu
- Find Facilities → locate hospitals, clinics & health centres
- Book & Manage Appointments
- Medication & Health Records
- Symptom Checker – simple health guidance
- Vaccination schedule & history
- Health Education & Prevention Tips
- Emergency access & contacts
- List of available services
- About the system

 3. Design Details
- Optimized for mobile phones
- Clean Green + White health theme
- Simple icons & large buttons
- Easy reading and navigation
- Works well in low‑literacy settings

 Database Purpose
This supports your clinic database assignment:
- Tables include: Users, Facilities, Staff, Appointments, Medicines, Vaccines, Records
- Easy to build in phpMyAdmin / MySQL
- 
 How to Use
1. Open → Welcome screen
2. Register → create account
3. Login → enter Dashboard
4. Use any feature from menu
5. Save / view records anytime

 Project Benefits
- Reduces long queues
- Short travel distance
- Stores data safely
- Gives reliable health education
- Helps emergency response
